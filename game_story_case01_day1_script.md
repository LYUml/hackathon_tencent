# 《画皮》第一案：第一天白天案发现场侦察台本

## 定位

本案发生在现代城市中的薛氏剧团旧址。玩家身份是受匿名委托而来的侦探，公开身份是民俗犯罪顾问，私下调查一封没有署名的短信：

> 别让民国那场火，再烧一次。

核心主题是“现实，是历史的重演”。这里的“重演”不是超自然，而是一种犯罪方法：凶手利用剧团排练、道具权限、锣鼓节奏、门锁结构和众人的沉默，把九十三年前的民国火案重新搭建到现代。

## 民国旧案

九十三年前，薛氏戏班在同一座戏院演出《断面》。当晚锣鼓点提前三拍，后台起火，青衣演员失踪，净角演员被认定救火失败，班主用“火烛意外”结案。旧案里有三个未被解释的问题：

- 锣鼓点为什么提前。
- 后台小屋为什么呈现“屋内反锁”。
- 失踪青衣为什么留下写着“勿忘”的旧戏票。

现代复排《断面》时，上述三件事再次出现。

## 第一案现代事件

第一天白天，玩家进入戏班后台。起初这是调查阶段，不是夜晚演出。玩家刚完成自言自语的身份导入，就听见侧台传来短促锣声。净角演员钟铁面在盔头与道具陈列室倒下，众人称他“被旧案找上了”。

钟铁面没有立刻死亡，可以设定为昏迷重伤，方便两天 Demo 保留审问和后续反转。他倒下时戴着半张烧焦脸谱，地上有旧案同款戏票。现场看起来像民国火案的复制，但侦探必须证明这是人为布置。

## 第一现场

场景：戏曲道具与盔头陈列室。

玩法方式：弹丸论破式侦察。玩家点击现场异常区域，每调查一个区域获得一条证据条。证据条进入背包，可用于后续盘问、观皮、判定。

### 调查点与证据

1. 倒地位置
   - 获得证据：倒地痕迹
   - 发现：地板拖痕从门口向屋内延伸，说明钟铁面并非在最终倒地位置受到袭击。

2. 侧门门闩
   - 获得证据：反扣门闩
   - 发现：门闩像从屋内合上，但木屑集中在门外，说明外部有人复原反锁假象。

3. 脸谱盒
   - 获得证据：新鲜香灰
   - 发现：烧焦脸谱被故意放在最上层，旁边香灰仍有温度，旧案气味是现代伪造。

4. 锣鼓公告板
   - 获得证据：错位锣点
   - 发现：锣鼓点被提前三拍，正好盖住门闩落下和人体倒地的声音。

5. 化妆台药瓶
   - 获得证据：嗓子药瓶
   - 发现：润喉药瓶底部有镇静药粉，钟铁面倒下前喝过水。

6. 幕布滑轨
   - 获得证据：吊景细线
   - 发现：细线绕过滑轨，方向通向二楼雅座，可远距离牵动门闩和幕布。

## 第一轮嫌疑人结构

沈青衣：
现代旦角。知道旧案细节，对“青衣勿忘”反应异常。她可能隐藏旧案受害者后人的身份，但缺少修改监控和管理道具的权限。

阿喜：
丑角兼杂务。承认摸过后台钥匙，行为可疑，但动机更像小偷小摸。他可以成为第一轮误导对象。

薛万山：
剧团班主。掌握复排流程、道具出借、监控权限和后台钥匙。第一天不直接定罪，但他的权限链必须被玩家逐步看到。

无脸人：
旧照和雅座阴影中的人物。第一天只作为“被利用的旧案证人”出现，不要立刻解释清楚。

## 侦探独白节奏

进入后台后：

“我推开侧门，后台的木地板发出很轻的一声响。戏服挂在暗处，盔头箱半开着，舞台那边有人试了一下锣点，又很快停住。”

听见事故后：

“如果这是诅咒，诅咒也太懂排练表了。它知道什么时候敲锣，知道谁会喝水，知道哪一道门闩能从外面被复原。”

收集三条证据后：

“历史不会自己走回原位。有人替它摆好了门、线、声音和恐惧。”

收集六条证据后：

“现场已经说完了它能说的话。接下来该让人说话了。”

## 第一轮判定

推理问题：

“钟铁面倒下后，为什么现场看起来像民国旧案重演？”

正确证据链：

错位锣点 → 嗓子药瓶 → 反扣门闩 → 吊景细线 → 新鲜香灰

结论：

有人先让钟铁面摄入镇静药，利用提前锣点遮掩现场声音，再通过滑轨细线制造反锁假象，最后用烧焦脸谱和香灰伪装成民国旧案重演。

错误判定：

如果玩家只提交“旧戏票 / 民国旧照 / 无脸人传闻”，扣一颗心。提示：这些是旧案关联，不足以证明现代作案手法。

## Demo 到此处的目标

两天 Demo 不需要完整结案。第一天白天应完成：

- 玩家身份明确。
- 现代与民国旧案关系明确。
- 后台案发现场可调查。
- 六条证据可收集。
- 证据背包、心数、存档系统可用。
- 收齐证据后开放目击者盘问和第一轮推理判定。
