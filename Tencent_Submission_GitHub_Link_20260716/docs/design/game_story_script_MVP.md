# 《画皮》完整剧情台本与两日 Demo 截取

版本：完整剧情规划 + 两日 Demo 制作版  
类型：本格推理 / 现代剧团 / 轻恐怖 / 观皮解谜  
核心诡计：现实正在重演历史，但不是鬼怪循环，而是有人刻意复刻当年的火案现场。

## 0. 时间线设定

- 现在：现代社会。玩家进入的是现代“薛氏剧团”，剧团正在复排旧戏《断面》。
- 过去：案件发生在民国二十年。旧“薛氏戏班”所在剧院失火，留下三死两失踪的旧案。
- 重演：现代剧团复排当晚，有人利用民国旧案档案、旧道具和后台权限，把现代后台布置成当年火灾夜。
- 玩家任务：不是侦破现代火灾，而是在现代重演即将发生前，查清民国旧案里“没有脸的人”到底是谁，以及谁在推动这场重演。

## 0.5 制作范围

两天是制作周期，不是剧情长度。当前两日 Demo 只截取完整剧情的第一章：玩家入场、建立民国旧案、收集第一批证据、完成第一次判定、揭开沈青衣第一张皮，并把后续目标推向“封闭后台小屋”。

### 必做

- 2 个场景：戏曲道具与盔头陈列室、排练室。
- 4 个角色：旦角沈青衣、丑角阿喜、班主薛万山、净角周岱。
- 8 个线索：旧戏票、脸谱盒、门闩划痕、阿喜口供、演出流程表、被改动的排练表、道具库出借记录、监控缺失片段。
- 证据条背包：线索以横向条状证据卡显示，用于推理和观皮判定。
- 五心血量：每天 5 颗心，错误判定一次损失 1 颗。
- 1 次小推理：证明“历史重演”是人为复刻。
- 1 次观皮：揭开旦角的第一层秘密。
- 1 个结尾：班主发现玩家查到不该查的东西。

- 来源地点
- 关键词

视觉方向：
- 默认：旧纸色横条。

## 1. 故事总览

玩家是一名年轻私家侦探，收到匿名委托邮件，调查民国二十年薛氏戏班旧剧院火灾失踪案。为了接近真相，玩家伪装成新媒体实习生，进入现代薛氏剧团拍摄复排幕后花絮。

剧团今晚要复排《断面》。玩家很快发现，这出戏正是当年火灾发生前最后一场戏。戏曲道具与盔头陈列室里的物件、门闩痕迹、演出流程，都和旧案记录一一对应。

玩家起初以为是剧团闹鬼，后来通过线索推出：有人正在人为复刻当年的火灾夜。旦角沈青衣并非凶手，而是当年火案的知情人。真正知道最多、也最有能力复刻现场的人，是班主薛万山。

Demo 结束时，玩家只揭开第一层真相：当年失踪的人可能还活着，而且“没有脸”。

## 1.5 完整剧情章节规划

### 第一章：第一张皮

制作状态：两日 Demo 必做。

剧情目标：让玩家明确三件事：自己身处现代，调查的是民国旧案；现代剧团正在复刻民国火案；沈青衣不是凶手，而是知情者。

主要场景：戏曲道具与盔头陈列室、排练室。

核心证据：旧戏票、脸谱盒、门闩划痕、演出流程表、被改动的排练表、道具库出借记录、监控缺失片段。

章节结尾：沈青衣透露“没有脸的人”仍在剧场里，班主发现玩家正在接近真相。

### 第二章：封住的小屋

制作状态：后续扩展。

剧情目标：从“谁在复刻旧案”推进到“旧案幸存者是否还活着”。

玩家根据沈青衣提供的信息，试图进入后台最深处被封住的小屋。薛万山阻止玩家靠近，并把阿喜偷钥匙的事公开，试图把所有嫌疑推到阿喜身上。

小屋调查内容：
- 旧药瓶：标签被撕掉，但药名能看出长期用于烧伤护理。
- 换下的纱布：不是新伤，是多年旧伤的护理痕迹。
- 剪碎的证件：照片部分被烧掉，只剩姓名偏旁。
- 不照人的镜子：镜面被黑布包住，旁边写着“勿照”。
- 旧戏服夹层：藏着一张民国火案后失踪人员名单。

本章推理：所谓“没有脸的人”不是鬼，而是民国火案幸存者。他被严重烧伤后失去原本身份，被薛家长期藏在剧团里。

结尾反转：小屋里没有人，只留下刚被搬走的生活痕迹。说明班主已经抢先一步转移了他。

### 第三章：谁寄出的委托

制作状态：后续扩展。

剧情目标：把玩家从单纯查案推向“谁在利用玩家”。

玩家发现匿名委托不是沈青衣发的，也不是阿喜发的。周岱虽然保留了记录，但他不敢主动请外人进来。委托人极可能就是“没有脸的人”本人，或者是一直替他送信的人。

关键矛盾：
- 匿名邮件里有现代监控截图，说明发信者能接触现在的后台。
- 邮件附件里有民国旧戏票扫描件，说明发信者也接触过旧案遗物。
- 委托金来自一个空壳文化公司，登记地址曾属于薛氏剧团旧仓库。

本章推理：玩家不是被偶然请来的侦探，而是被旧案幸存者或其保护者精确引入剧场。玩家的作用，是在正式演出前打破薛万山对真相的控制。

结尾反转：玩家收到第二封邮件，只有一句话：“不要相信第一个认罪的人。”

### 第四章：民国二十年的火

制作状态：后续扩展。

剧情目标：还原民国旧案，建立本格核心诡计。

旧案真相不是简单纵火。民国二十年，《断面》演出前，薛氏戏班内部发生身份替换：一名原本该登台的人被迫退场，另一人戴上他的脸谱和戏服替他上台。火灾发生后，死者身份因此被混淆。

关键旧案证据：
- 烧焦戏票背面的“青衣勿忘”，不是写给现代沈青衣，而是写给当年的青衣。
- 火案名单里的“两人失踪”，其实一人死亡、一人换名活下。
- 封门痕迹显示，后台门不是火起后变形卡住，而是在火起前被人为锁死。
- 当年的《断面》唱词被改过，改动处暗示了身份替换。

本章真相：当年有人利用戏曲扮相、脸谱和混乱火场完成身份替换。火不是为了杀所有人，而是为了烧掉能证明“谁是谁”的证据。

### 第五章：最后一场《断面》

制作状态：后续扩展。

剧情目标：让现代演出成为最后推理场。

薛万山宣布《断面》正式演出照常进行。台上唱的是戏，台下推进的是灭口。玩家需要在演出节点之间完成连续判定，用证据阻止“历史重演”变成真正的二次火案。

连续判定节点：
- 指出阿喜偷钥匙不等于他改流程。
- 指出周岱调道具是执行命令，不是主谋。
- 指出沈青衣说谎是为了保护幸存者，不是为了掩盖纵火。
- 指出薛万山拥有改流程、调道具、删监控和转移小屋住客的完整权限。

最终观皮目标：薛万山。

最终揭示：薛万山不是民国火案原始凶手，但他继承并维护了薛家的谎言。现代重演不是纪念，而是筛查还有谁知道真相，并把最后的人证从世界上抹掉。

结局方向：
- 普通结局：玩家阻止正式演出事故，救出“没有脸的人”，但旧案公开证据不足。
- 真结局：玩家完成全证据链，证明民国火案和现代重演的因果关系，旧案重启调查。
- 坏结局：判定失败，薛万山完成转移，正式演出结束后所有人都声称“什么也没发生”。

## 6. NPC 对话台本

对话系统参考弹丸论破的调查段：点击角色后进入话题菜单。每个角色有“普通话题”“追问话题”“证据出示”三层。玩家不用一次把话问完，可以退出后继续探索；获得新证据后，旧角色会出现新反应。

UI 建议：
- 左侧显示角色立绘。
- 右侧显示对话框。
- 底部显示话题按钮。
- 已问过的话题变灰，但可重复查看。
- 解锁的新话题用暗红标记。
- 证据出示时弹出横向证据条背包。

对话状态：
- 平静：普通对话。
- 警戒：玩家问到旧案、后门、监控时触发。
- 破绽：玩家出示正确证据后触发，角色说出关键情报。
- 封口：角色拒绝继续回答，但留下新的调查目标。

周岱：

> 人怕归怕，不能一点后路都不留。

### 证据出示：监控缺失片段

### 周岱对话结束

> 你要真想查，就别只看谁碰过道具。

### 判定 UI

画面底部弹出证据条背包。

提示：

可选证据条：
- 旧戏箱里的旧戏票
- 沾灰的脸谱盒

正确组合：

- 道具库出借记录

增强正确组合：
- 演出流程表
- 被改动的排练表
- 道具库出借记录
- 监控缺失片段

错误判定反馈：
- 选择“旧戏票 + 脸谱盒”：心 -1。提示：“这能证明旧案和沈青衣有关，但不能证明今晚被人改动。”
- 选择“阿喜的口供”：心 -1。提示：“口供能提供方向，但不能单独证明复刻。”
- 选择“门闩划痕”：心 -1。提示：“门闩证明现场被布置过，但还缺少谁有能力安排这一切。”
- 指认沈青衣为主谋：心 -1。提示：“她被证据指向，但更像被复刻现场逼迫的人。”

系统旁白：

> 旧戏票证明《断面》和沈青衣有关。

若有门闩划痕：

若有阿喜口供：

> 阿喜听见的旧戏词，也许不是青衣在怀旧，而是有人逼她再次走进那一晚。

第一阶段结论：

> 它和民国二十年那场火有关。

第二阶段补充：

> 监控缺失的三分钟，刚好覆盖了有人进出后台的时间。

第二阶段结论：

> 历史不是自己重演。

> 是有人把每个人、每件道具、每一道门，都重新摆回当年的位置。

> 能改流程、调道具、删监控的人，不会是普通演员。
