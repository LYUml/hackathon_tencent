# HuaPi Unity Folder Map

This folder contains the game-specific assets for HuaPi. Unity-generated caches such as
`Library`, `Logs`, and `UserSettings` are intentionally outside this folder and can be
regenerated by Unity.

## Main Folders

- `Scenes/`
  - Unity `.unity` scenes.
  - Current playable scene: `Scenes/Main.unity`.

- `Art/`
  - Imported 2D art assets.
  - `Art/Backgrounds/`: location illustrations.
  - `Art/Characters/`: character portraits and standees.

- `Data/`
  - ScriptableObject gameplay data.
  - `Data/ExplorationScenes/Demo/`: hand-curated demo scene data used by the current build.
  - `Data/ExplorationScenes/Generated/`: auto-generated location data kept for later use.

- `Prefabs/`
  - Reusable UGUI prefabs.
  - `Prefabs/Panels/`: full-screen or major UI panels.
  - `Prefabs/Exploration/`: small reusable exploration UI modules.

- `Scripts/`
  - Runtime and editor C# scripts.
  - `Scripts/Core/`: UI framework.
  - `Scripts/Data/`: ScriptableObject data types.
  - `Scripts/Panels/Exploration/`: first-phase exploration system.
  - `Scripts/Editor/`: Unity editor tools and generators.

## Notes

- Keep `.meta` files with their assets when moving files. Unity references assets by GUID.
- Do not manually edit or commit `Library/`, `Logs/`, or generated `.csproj` files.
- The many generated exploration scene assets are intentionally stored under
  `Data/ExplorationScenes/Generated/` so they do not clutter the hand-authored demo data.
