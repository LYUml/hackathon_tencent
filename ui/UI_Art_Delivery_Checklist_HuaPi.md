# 《画皮》UI 美术需求与交付清单

基准：1920x1080，16:9 横屏。  
项目定位：现代侦探调查民国旧案，旧戏班、暗场、水墨、旧纸、脸谱、黑雾。  
制作范围：两天 Demo 只做第一章“第一张皮”，但 UI 要能支撑后续章节扩展。

## 0. 美术总原则

- UI 不要像现代软件面板，要像场景里的旧戏票、旧纸、墨迹、照片、字幕和档案。
- 尽量少用硬矩形白框；多用暗场渐变、旧纸边、墨迹遮罩、暗红印章。
- 文字可读性优先，恐怖氛围不能压过剧情阅读。
- 重要交互要有状态差异：默认、Hover、选中、已使用、禁用、错误。
- Demo 最核心体验是：调查、对话、拿证据、出示证据、推理扣心、观皮揭示。

主色建议：
- 极深黑 `#0a0808`
- 旧纸白 `#d8cfc0`
- 暗红 `#8b2020`
- 暗铜金 `#c9a96e`
- 墨灰 `#5a5a5a`

通用素材：
- 旧纸底纹
- 撕裂纸边
- 戏票纸条
- 暗红印章
- 墨迹扩散
- 黑雾遮罩
- 暗场渐变
- 可点击虚线/墨线

## 1. 两天 Demo 最高优先级

第一优先级，必须先完成：
1. 对话界面
2. 证据条背包
3. 判定 / 五心血量 UI
4. 观皮界面

第二优先级，Demo 需要但可简化：
1. 探索 HUD
2. 人物档案简版
3. Toast / 目标更新 / 错误提示

第三优先级，可最后 polish：
1. 主菜单
2. 暂停 / 设置
3. 完整人物关系档案

## 2. 对话界面

用途：支撑弹丸论破式调查对话。玩家点击角色后进入话题菜单，选择普通话题、追问话题，或打开证据条背包出示证据。

需要出图：
- 底部对白暗场渐变。
- 人物名牌。
- 话题菜单按钮。
- 话题已问过状态。
- 新解锁话题状态。
- 不可用话题状态。
- 证据出示入口。
- 继续提示。
- 重要对白强调样式。
- 当前说话人高亮 / 非当前人物压暗。

话题状态：
- 普通话题：旧纸色或暗金文字。
- 已问过：墨灰、低透明。
- 新解锁：暗红边缘或印章标记。
- 不可用：墨雾遮挡。
- Hover：暗铜金描边。
- 选中：暗红墨迹扩散。

对话状态：
- 平静：普通对白。
- 警戒：角色被问到旧案、后门、监控。
- 破绽：玩家出示正确证据后，角色说出关键情报。
- 封口：角色拒绝继续回答，但留下新调查目标。

美术备注：
- 不要做白框式对话框。
- 对白像舞台字幕从黑暗中浮现。
- 选项像漂浮旧纸签，不要现代按钮。
- 重要词如“民国二十年”“没有脸的人”“后台小屋”“《断面》”可用暗红强调。

## 3. 证据条背包

用途：参考弹丸论破“言弹”，但视觉改成旧戏票 / 旧纸条。用于对话出示、推理判定、观皮选择。

Demo 证据：
- 旧戏箱里的旧戏票
- 沾灰的脸谱盒
- 后台门闩划痕
- 阿喜的口供
- 演出流程表
- 被改动的排练表
- 道具库出借记录
- 监控缺失片段

需要出图：
- 横向证据条。
- 证据条小图标或分类标记。
- 证据详情纸页。
- 证据来源字段样式。
- 关键词字段样式。
- 新线索提示条。
- 关闭符号。

证据条状态：
- 默认：旧纸色横条。
- Hover：暗铜金描边。
- 选中：暗红墨迹高亮。
- 新获得：纸角亮起或盖章。
- 已读：纸色变暗。
- 关键证据：暗红印章。
- 可用于当前判定：纸角出现红线。
- 错误使用：短暂变黑，被墨吞回。
- 正确使用：纸角留下红线标记。

美术备注：
- 证据条要适合横向快速选择。
- 不要做成普通背包格子。
- 证据详情可以像侦探桌面上的旧纸、照片和戏票。

## 4. 判定 / 五心血量 UI

用途：推理和观皮错误时扣心。每天 5 颗心，错误一次损失 1 颗。

需要出图：
- 五心图标。
- 心破裂状态。
- 心熄灭状态。
- 剩 1 颗心危险提示。
- 错误判定墨迹吞回效果。
- 0 心调查失败画面。

状态：
- 5/5 满心
- 4/5
- 3/5
- 2/5
- 1/5 危险
- 0/5 调查失败

美术备注：
- 心不要做现代卡通红心。
- 可做成暗红纸符、脸谱碎片、血色墨点、戏票缺口。
- 错误扣心时：一颗心裂开、褪色、熄灭。
- 1 心危险时：画面边缘轻微压暗，暗红墨迹浮现。

失败文案画面：
- “调查失败。”
- “班主发现你的身份异常，你被请离后台。”
- “从当天开始重新调查。”

## 5. 观皮界面

用途：特色界面。玩家用 3 条证据揭开角色黑雾，看破第一张皮。

Demo 目标：沈青衣。  
正确组合：旧戏票 + 脸谱盒 + 演出流程表。

需要出图：
- 中央画像区域。
- 黑雾 / 墨雾遮罩。
- 证据槽纸签。
- 证据拖拽或选中轨迹。
- “观皮”提交按钮 / 字样。
- 正确揭示效果。
- 错误吞回效果。
- 完全揭示高光。

状态：
- 未选择证据。
- 证据 Hover。
- 证据选中。
- 证据槽已放入。
- 正确匹配。
- 错误匹配。
- 局部揭示。
- 完全揭示。

美术备注：
- 不要做成普通合成界面。
- 玩家要感觉是在黑雾中的画像前“看破一层皮”。
- 正确时墨雾从局部散开，露出记忆碎片或人物真相。
- 错误时证据条被墨雾吞回，同时扣 1 心。

## 6. 探索 HUD

用途：玩家在“戏曲道具与盔头陈列室”“排练室”探索时使用。

需要出图：
- 地点文字样式。
- 当前目标提示条。
- 线索入口。
- 档案入口。
- 暂停入口。
- 调查热点样式。
- 人物可对话入口样式。

状态：
- 可调查
- 已调查
- 关键线索
- 可对话
- 已对话
- 不可用

美术备注：
- HUD 要轻，优先保留场景画面。
- 热点不要太游戏化，像暗铜虚线、墨迹圈、轻微红线。
- 当前目标可以是一行墨字浮现，不要大任务框。

- 可疑点区。

- 英文副标题样式。
- 菜单文字：开始游戏、继续游戏、设置、退出。

状态：

- 选中

- 音量 Slider。
- 文字速度 Slider。
- 亮度 Slider。
- 开关 Toggle。
- 确认弹窗。

状态：
- 默认
- Hover
- 选中
- 禁用
- 确认 / 取消

美术备注：
- 像暗场中的烛光文字。

## 10. 弹窗和提示

需要出图：
- 获得线索 Toast。
- 人物档案更新 Toast。
- 目标更新提示。
- 章节标题卡。
- 地点切换标题卡。
- 错误判定提示。
- 退出确认。

状态：
- 出现
- 停留
- 消失
- 重要提示
- 普通提示

美术备注：
- Toast 可以是旧纸条滑入。
- 目标更新可以是一行墨字浮现。
- 错误提示用暗红墨迹一闪，不要长时间打断。
- 确认弹窗尽量用全屏压暗 + 文字，不要独立白框。
