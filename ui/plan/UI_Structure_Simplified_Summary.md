# 《画皮》UI汇总结构图（简化版）

用途：给美术快速确认页面数量、弹窗数量、页面跳转关系和每页必要功能。

## 页面数量

主页面共 9 个：

1. 标题页
2. 开场 Intro
3. 白天探索 HUD
4. 对话页
5. 证据背包
6. 人物档案
7. 观皮界面
8. 推理判定
9. 暂停 / 设置

弹窗共 5 类：

1. 操作指南弹窗
2. 证据详情弹窗
3. 存读档弹窗
4. 目标 / 获得线索提示
5. 扣心 / 失败弹窗

## 核心连接

标题页 -> 开场 Intro -> 白天探索 HUD。

白天探索 HUD 可以进入：

- 对话页
- 证据背包
- 人物档案
- 暂停 / 设置

对话页可以调用：

- 证据背包
- 证据详情弹窗

证据背包可以被三处调用：

- 对话出示证据
- 观皮选择证据
- 推理判定提交证据

观皮界面和推理判定都会关联心数系统：

- 错误一次扣 1 颗心
- 5 颗心归零进入失败弹窗
