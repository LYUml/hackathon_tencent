# 《画皮》解密关卡设计草案（参考纸嫁衣式体验）

## 设计原则

参考方向是“纸嫁衣”系列的民俗悬疑解密节奏：玩家在多个房间来回移动，观察环境里的符号、器物、顺序和禁忌，通过道具组合打开下一层线索。但本作保持现代本格推理定位，所有“仪式感”最终都要能落回人为布置、旧案复刻、权限链和时间线。

核心体验：

- 画面里藏线索，不靠大字条提示。
- 道具不是收集品，而是解锁另一个场景/机关的钥匙。
- 民俗符号先制造不安，之后被推理还原成作案手法。
- 每个谜题都要服务“现实，是历史的重演”。

## 第一章：后台案发现场

### 谜题 1：错位锣点

场景：戏曲舞台、排练室、道具陈列室。

玩家获得：

- 排练室公告板上的锣鼓谱。
- 舞台边的旧唱机节拍。
- 道具陈列室里倒地时刻的记录。

解法：

玩家需要把公告板上的锣点顺序和旧唱机节拍对照，发现第三拍被提前。正确顺序为“慢、急、停、急、慢”。输入到舞台锣鼓机关后，舞台暗格打开。

解锁：

- 获得“改过的锣鼓谱”。
- 证明倒地声被锣鼓掩盖。

### 谜题 2：脸谱五色

场景：道具陈列室。

玩家获得：

- 五张脸谱：红、黑、白、蓝、金。
- 民国旧照里演员站位。
- 戏箱里一张残缺戏单。

解法：

按照旧戏单的生旦净丑顺序摆放脸谱，但白脸不能放在台前，因为旧照中白脸站在幕后。正确顺序：红、金、黑、蓝、白。

解锁：

- 脸谱盒夹层打开。
- 获得“新鲜香灰”和“旧戏票残页”。

### 谜题 3：反扣门闩

场景：侧台门帘、道具陈列室、上座雅座。

玩家获得：

- 幕布滑轨细线。
- 门闩木屑。
- 雅座扶手上的铜环。

解法：

玩家需要在地图上确认“雅座 → 滑轨 → 侧门”的直线关系，再把细线绕过滑轨模型，模拟从门外拉动门闩。交互方式可以做成简单拖线谜题。

解锁：

- 获得“门外反锁复原图”。
- 证明密室不是密室。

### 谜题 4：润喉药瓶

场景：排练室、后台茶桌、道具陈列室。

玩家获得：

- 三个杯子。
- 润喉药瓶。
- 演员喝水顺序。

解法：

杯底有不同残渍。玩家按照排练室座位表对应杯子，发现钟铁面喝的是贴有“护嗓”纸条的杯子，但杯底残留镇静药粉。

解锁：

- 获得“嗓子药瓶”证据。
- 开启对阿喜和沈青衣的盘问。

## 第二章：夜晚复排

### 谜题 5：戏箱封条

场景：老旧戏箱堆放角。

玩法：

四个戏箱分别贴着“生、旦、净、丑”，但封条年份与演员行当不对应。玩家需要根据民国旧照和现代演员档案重新排序。

解锁：

- 找到旧案遗物。
- 知道当年失踪青衣不是死者，而是被替换了身份。

### 谜题 6：灯笼影子

场景：戏曲舞台、上座雅座。

玩法：

玩家调整三盏灯笼角度，让舞台幕布上的影子重合成“无脸人”的轮廓。完成后发现影子来自机械牵线，不是鬼影。

解锁：

- 获得“牵影结构图”。
- 破除无脸人传闻。

## 第三章：判定系统结合

每个谜题完成后，获得一条“证据条”。推理判定时不是随便提交，而是要求顺序：

1. 嗓子药瓶：让钟铁面失力。
2. 错位锣点：遮蔽倒地声音。
3. 吊景细线：远程操控门闩。
4. 新鲜香灰：伪造旧案气味。
5. 旧戏票残页：把现代案嫁接到民国旧案。

错误提交扣一颗心。提示方式不直接给答案，只提示“这是旧案关联，不是现代手法”或“你证明了动机，但还没有证明操作方法”。

## Demo 优先级

两天 Demo 建议先做：

1. 脸谱五色：最适合视觉表现，也符合新道具陈列室图。
2. 反扣门闩：最贴合本格推理，能体现“密室破解”。
3. 错位锣点：作为推理判定前的关键证据。

暂缓：

- 灯笼影子需要额外动画或可调光源。
- 戏箱封条需要更多旧案文本。
